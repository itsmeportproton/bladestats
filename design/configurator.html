Release log

0.1.0 2026-07-20
  • Frame timing without injection. FPS, frame time and low percentiles read from the graphics kernel's own events. Nothing is loaded into the game.
  • Direct3D and Vulkan alike. Presents are counted at the kernel, so Vulkan titles report the same as Direct3D ones.
  • Per-core load and clocks, with exact processor and graphics card names as Device Manager spells them.
  • Memory speed read from firmware rather than through WMI.
  • Overlay composited on the GPU, so transparency costs nothing on the processor.
  • Fixed: VRAM showed 0.0 GB. It was reporting bladestats' own video memory instead of the whole system's.
  • Fixed: every graphics reading sat at zero on some machines, from a case-sensitive adapter match.
  • Fixed: frame rate stayed blank when launched without administrator rights, with no explanation on screen.
Known limits
  • Borderless only. Exclusive fullscreen cannot be drawn over, so the overlay hides itself there.
  • Graphics temperature and power need a vendor library; AMD and Intel are not read yet.
  • Processor power is an estimate, always marked with a tilde.
  • Memory power is never shown. Consumer boards have no sensor for it.
