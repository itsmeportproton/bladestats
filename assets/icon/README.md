# Icon

`bladestats.ico` is generated by `make-icon.py`, which needs nothing but a Python install:

```sh
python assets/icon/make-icon.py
```

It is committed alongside the script so a build needs neither Python nor a network. Change the
numbers at the top of the script and run it again rather than editing the `.ico`.

The mark is the overlay's per-core load strip, which is the most recognisable thing bladestats
draws and still reads as four bars at sixteen pixels, where a wordmark would be a smear.

`crates/bs-config/bladestats.rc` compiles it into the executable as resource 1. The taskbar,
Explorer and Alt-Tab read it from there; the notification area asks for the same resource by
number, so there is one icon rather than two that can drift apart.
